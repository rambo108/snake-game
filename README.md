# 🐍 Minecraft Snake

A Minecraft-themed take on the classic Snake game — built entirely in a single `index.html` file. No build tools, no dependencies, no excuses. Just open it in a browser and play.

## ✨ Features

- **Creeper Snake** — your snake sports a pixel-art Creeper face on its head
- **Dirt-block world** — the board uses a Minecraft dirt-brown palette with stone-gray grid lines
- **Minecraft apple** — collect pixel-art apples to grow and score points
- **Pickup sounds** — crunchy square-wave audio on every apple, powered by the Web Audio API
- **Progressive difficulty** — speed increases every 5 apples eaten
- **High score persistence** — your best score is saved to `localStorage`
- **Pause menu** — spacebar toggles a Minecraft-style pause overlay
- **Mobile controls** — on-screen touch buttons for playing on phones and tablets
- **Zero dependencies** — one HTML file, no frameworks, no bundlers

## 🎮 How to Play

1. Open `index.html` in any modern browser
2. Press any movement key or tap a direction button to start

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Move | Arrow keys / WASD | Touch buttons |
| Pause / Resume | Spacebar | — |
| Restart (after game over) | Any movement key | Tap a direction |

## 🛠 Tech Stack

- Vanilla HTML / CSS / JavaScript
- Canvas API (game rendering)
- Web Audio API (sound effects)
- `localStorage` (high score persistence)
- That's it. No dependencies.
